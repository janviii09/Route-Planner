# AI Route Planner

This project implements an AI Route Planner with multiple graph algorithms and a decision logic to choose the best algorithm dynamically.

## Features
- Graph Data Structure (Adjacency List)
- Disjoint Set Union (DSU) for cycle detection and connectivity checks
- Dijkstra's Algorithm (Shortest Path)
- A* Search (Shortest Path with Heuristics)
- BFS (Emergency Routing / Minimum Stops)
- Bellman-Ford (Handling Negative Weights)
- Kruskal's Algorithm (Minimum Spanning Tree)
- Decision Logic Router

## Running
```bash
cd route-planner
python main.py
```
